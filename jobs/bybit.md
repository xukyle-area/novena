# bybit

## Java-P2P

### 核心职责
1. Java 核心开发： 负责高并发订单处理中心、支付结算逻辑及 IM 消息引擎的研发，编写高质量、可维护的 Java 代码。
2. 架构优化与演进： 针对现有系统的技术债进行重构，优化 JVM 性能、数据库索引及缓存策略，解决分布式环境下的数据一致性问题。
3. AI 落地实践： 参与 AI 导购项目的架构设计，利用 Spring AI 或相关框架集成大模型能力，实现智能推荐、自动化客服等场景的落地。
4. 系统稳定性： 负责核心链路的监控与排障，应对大促或突发流量下的系统稳定性挑战。
### 职位要求
1. 技术背景： 3-5 年 Java 开发经验，底层功底扎实，深入理解 JVM 内存模型、垃圾回收机制及多线程并发编程。
2. 框架能力： 精通 Spring Boot、Spring Cloud、MyBatis 等主流框架，理解其核心原理及扩展点。
3. 领域经验（满足其一）：
    - 交易流转： 熟悉分布式事务、状态机设计，有处理复杂金融级订单或结算系统的经验。
    - 消息通讯： 熟悉 Netty 网络编程，对 WebSocket/MQTT 等协议有深入理解，具备百万级并发推送经验。
4. 中间件： 熟练运用消息队列（RocketMQ/Kafka）处理异步解耦，具备丰富的 MySQL 调优及 Redis 分布式锁应用经验。
