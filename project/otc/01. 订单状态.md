# 一、状态分类

根据图中的颜色可以分为三类：

| 类型       | 状态                                                                   |
| -------- | -------------------------------------------------------------------- |
| 初始状态（蓝色） | `PENDING_NEW`                                                        |
| 中间状态（黄色） | `PENDING_FAILED` `NEW` `CONFIRMED` `PARTIAL_FILLED` `PENDING_CANCEL` |
| 终态（绿色）   | `SETTLED` `FILLED` `PARTIAL_SETTLED` `CANCELED` `FAILED`             |

---

# 二、状态定义

## 1️⃣ PENDING_NEW

**含义：**

订单刚创建，正在 **锁定用户资产**。

**触发场景**

用户下单后，系统调用资金系统锁定资产。

**可能流转**

```
PENDING_NEW -> NEW
PENDING_NEW -> PENDING_FAILED
```

---

## 2️⃣ PENDING_FAILED

**含义：**

第一次 **锁定用户资产失败**，系统准备 **重试锁定资产**。

**可能流转**

```
PENDING_FAILED -> NEW     （锁定成功）
PENDING_FAILED -> FAILED  （锁定再次失败）
```

---

## 3️⃣ FAILED  （终态）

**含义：**

订单创建失败，原因是 **资产锁定失败**。

**特点**

* 订单不会再继续执行
* 用户资产未锁定
* 订单结束

---

## 4️⃣ NEW

**含义：**

资产锁定成功，订单 **已创建成功**，等待发送给 LP。

**触发事件**

系统向 LP (Liquidity Provider) 下单。

**可能流转**

```
NEW -> CONFIRMED
```

---

## 5️⃣ CONFIRMED

**含义：**

LP 已确认订单，并返回 **成交报告（execution report）**。

此时订单已经进入 **撮合或成交阶段**。

**可能流转**

```
CONFIRMED -> PARTIAL_FILLED
CONFIRMED -> SETTLED
CONFIRMED -> FILLED
CONFIRMED -> PENDING_CANCEL
```

---

## 6️⃣ PARTIAL_FILLED

**含义：**

订单 **部分成交**，但仍有剩余数量未成交。

同时系统已对 **已成交部分发起结算**。

**可能流转**

```
PARTIAL_FILLED -> PARTIAL_FILLED   （继续部分成交）
PARTIAL_FILLED -> SETTLED          （最终全部结算）
PARTIAL_FILLED -> PENDING_CANCEL
```

---

## 7️⃣ SETTLED  （终态）

**含义：**

订单 **已完全结算完成**。

特点：

* 已完成成交
* 已完成资金结算
* 订单生命周期结束

---

## 8️⃣ FILLED  （终态）

**含义：**

订单 **全部成交**，并发起 **net 结算**。

通常代表：

* 成交完成
* 后续由 net 结算系统处理

---

## 9️⃣ PENDING_CANCEL

**含义：**

用户发起 **取消订单请求**。

系统正在处理取消逻辑。

**注意**

某些状态 **不允许取消**。

**可能流转**

```
PENDING_CANCEL -> PARTIAL_SETTLED
PENDING_CANCEL -> CANCELED
```

---

## 🔟 PARTIAL_SETTLED （终态）

**含义：**

订单 **部分成交并已结算**，剩余未成交部分被取消。

特点：

* 一部分成交
* 一部分取消
* 订单结束

---

## 1️⃣1️⃣ CANCELED （终态）

**含义：**

订单 **完全未成交，被取消**。

特点：

* 没有发生结算
* 资产解锁
* 订单结束

---

# 三、完整状态机（简化）

```
PENDING_NEW
   │
   ├──> NEW
   │      │
   │      └──> CONFIRMED
   │               │
   │               ├──> PARTIAL_FILLED
   │               │        │
   │               │        └──> SETTLED
   │               │
   │               ├──> FILLED
   │               │
   │               └──> PENDING_CANCEL
   │                         │
   │                         ├──> PARTIAL_SETTLED
   │                         └──> CANCELED
   │
   └──> PENDING_FAILED
            │
            ├──> NEW
            └──> FAILED
```
